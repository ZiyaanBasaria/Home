How to set up the chess board

The first thing that you should know before you start playing is how to set up the chess board. Here is a picture of how to set up the chess board:

As you can see the rooks (castles) are on the two corners of you side. Beside them are the knights (horses) then the bishops. The Queen goes on the same colour square that she is and the last square is left for the King. The pawns line up on the seccond row.
